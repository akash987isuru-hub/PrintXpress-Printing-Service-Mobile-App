# PrintXpress - Printing Service Mobile Application

PrintXpress is an Android mobile application developed for a printing service business. The app allows customers to browse printing product categories, upload designs, add items to cart, place orders, complete payment details, track order status, and communicate with customer support. It also includes an admin side for managing products, categories, promotions, customer orders, customer details, revenue, and support queries.

## Project Overview

This project was developed as a practical mobile application to demonstrate Android development, Firebase integration, user authentication, CRUD operations, order management, and admin dashboard functionality.

## Main Features

### Customer Features

- Customer registration and login
- Home page with product categories and featured products
- Product search and product details pages
- Cart management
- Design upload option
- Delivery details page
- Order summary and payment page
- Order tracking
- Customer support queries and admin replies
- Guidelines and about pages

### Admin Features

- Admin dashboard
- Manage product categories
- Manage products
- Review uploaded designs
- Update order status
- Manage promotions
- View customers
- View revenue details
- Reply to customer support queries

## How to Run the Project

1. Install Android Studio.
2. Clone or download this repository.
3. Open the `PrintXpress` folder in Android Studio.
4. Create a Firebase project.
5. Enable Firebase Authentication, Cloud Firestore, and Firebase Storage.
6. Download your own `google-services.json` file from Firebase.
7. Place the file inside the `app/` folder.
8. Sync Gradle.
9. Run the app on an Android emulator or physical Android device.

## Important Note

The `google-services.json` file is not included in this public repository for security and privacy reasons. Users should create their own Firebase project and add their own Firebase configuration file.
